For Apple Silicon · macOS 15+

Take control of your Mac's fans.

A polished, open-source fan controller for Apple Silicon. Smart cooling modes, live charts, a Liquid Glass menu-bar UI, and a privileged daemon that just works.

Install with Homebrew View on GitHub
$ brew install --cask Juanipis/tap/fanctl

What it does

Five thoughtful cooling modes plus full manual control, all from your menu bar.

⚙️

Smart modes

Auto, Silent, Cool, Performance, Custom, Manual. Each runs a temperature → RPM curve evaluated 2× per second by a privileged background daemon.

📈

Live charts

RPM and temperature sparklines update in real time. History persists across restarts. Hottest sensor is always one glance away.

🎚

Custom curves

Drag points on a chart to define your own curve. Pick any SMC sensor as the driver — CPU, GPU, memory, ambient, whatever.

🛡️

Watchdog

Dead-man heartbeat plus 95 °C thermal panic. The helper falls back to macOS auto if anything goes wrong. You can never get stuck.

🔄

Auto-updates

Built-in Sparkle: signed, daily check, one-click apply. Or update via Homebrew. Or manually, your choice.

🆓

Free & open

MIT-licensed Swift + IOKit. No third-party kernel extensions, no telemetry, no subscriptions. Read the source, file a bug, send a PR.

Modes

Each curve maps the hottest sensor (or one you pick) to fan RPM as a fraction of the fan's range.

⚡ Auto
macOS owns the fans. Default. Safe.
🌙 Silent
50°C→0% · 75°C→50% · 90°C→100%
Quietest first. Only ramps when really hot.
❄️ Cool
40°C→0% · 60°C→50% · 75°C→100%
Keeps the chassis cold. Fan starts earlier.
⚡ Performance
35°C→50% · 50°C→100%
Aggressive cooling for sustained load.
✏️ Custom
your curve · your sensor
Edit in Preferences. Persists across reboots.
🎚 Manual
Drag the slider. Min / Mid / Max presets too.

Sandboxed UI, privileged helper, deliberate trust boundary.

Install

Three options. Pick whichever you like.

Homebrew

brew install --cask Juanipis/tap/fanctl

curl

curl -fsSL https://raw.githubusercontent.com/Juanipis/fanctl/main/scripts/install.sh | bash

Manual

Grab the latest release, unzip, drop into /Applications, run xattr -dr com.apple.quarantine /Applications/FanCtl.app, open it.

First run

  1. Click the fan icon in your menu bar.
  2. Click Install Helper. macOS will redirect you to System Settings → General → Login Items & Extensions → Background.
  3. Toggle FanCtl ON. Click Retry. The hero card lights up.
